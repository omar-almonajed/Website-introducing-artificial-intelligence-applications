# AI Apps Project  

مشروع ويب بسيط لإدارة واستعراض تطبيقات الذكاء الاصطناعي.  
يتيح للمستخدمين إضافة تطبيقات جديدة، عرضها، وإعادة تهيئة المشروع للوضع الافتراضي عبر **LocalStorage** في المتصفح.  

---

## هيكل المشروع
- **home.html** → الصفحة الرئيسية (تتضمن أسماء الطلاب وأرقامهم).  
- **aboutus.html** → صفحة معلومات التواصل والفريق.  
- **apps.html** → صفحة عرض التطبيقات.  
- **add_app.html** → صفحة إضافة تطبيق جديد.  
- **style.css** → تنسيقات المشروع.  
- **myFunctions.js** → الدوال والوظائف الجافاسكربتية.  

---

## طريقة الاستخدام
1. افتح الملفات محليًا عبر المتصفح أو ارفع المشروع على منصات مثل **GitHub Pages** أو **Netlify**.  
2. **قبل التسليم:**  
   - عدّل أسماء الطلاب وأرقامهم في `home.html`.  
   - حدّث معلومات التواصل في `aboutus.html`.  
3. لإضافة تطبيق جديد:  
   - افتح `add_app.html`.  
   - املأ النموذج واضغط "إرسال".  
   - سيتم تحويلك إلى `apps.html` حيث يظهر التطبيق الجديد.
